# Polymarket Toolkit

Open-source tools for Polymarket prediction market traders.

## Skills

### polymarket-profile

Input any Polymarket address → get a complete trading profile with PnL, win rate, current positions, category distribution, and top trades.

All data from public Polymarket APIs. No API key, no local database, no setup required.

**Usage**: Install the skill in your AI agent (OpenClaw / Claude Code), then ask it to profile any address.

```
Profile this Polymarket address: 0x...
```
